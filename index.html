Ubuntu终端下翻墙

  某一天我需要爬取Twitter下的数据，发现光浏览器里能访问twitter.com是不行的，必须要在终端下能访问，于是就在Linux安装了shadowsocks和proxychains实现终端翻墙。

proxychains安装：

git clone https://github.com/rofl0r/proxychains-ng.git
cd proxychains-ng
./configure
make & make install
cp ./src/proxychains.conf /etc/proxychains.conf
cd .. && rm -rf proxychains-ng

proxychains.conf配置：

vim /etc/proxychains.conf
